# カルガモカウンター

## ■ プロジェクト概要
大型ダンプトラックの作業回数（積み下ろし）を記録するためのIoTシステムです。

ボタンを押すことで作業回数と時刻を自動で記録し、データを保存・可視化します。

---

## ■ 目的
- 手書きの作業記録をなくす
- 作業回数を正確に記録する
- 業務のデジタル化

---

## ■ システム構成
- ATOM Lite（ボタン入力）
- Webアプリ（画面表示）
- Google Sheets（データ保存）

---

## ■ 仕組み
1. ボタンを押す
2. 時刻と回数を記録
3. データを送信
4. スプレッドシートに保存
5. Web画面で表示

---

## ■ 使用技術
- ESP32（ATOM Lite）
- HTML / JavaScript
- Google Apps Script
- GitHub

---

## ■ 役割分担
- hardware：ボタン・デバイス制御
- backend：データ保存API
- frontend：画面表示
